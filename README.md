NewsLens — AI Powered News Summarizer

NewsLens is an AI-powered news intelligence platform that fetches the latest global news and instantly generates concise summaries using modern LLM APIs. The application provides a clean interface where users can browse trending news by category or summarize any article by pasting text or providing a URL.

The goal of this project is to make news faster to consume, easier to understand, and more accessible through AI-generated insights.
